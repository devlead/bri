## Parameters

| Name                                | Type                                                    | Required                                                                                   | Description                                                                                     |
|-------------------------------------|---------------------------------------------------------|--------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| `name`                              | `secureString`                                          | Yes                                                                                        | Name of animal                                                                                  |
| `parent`                            | `string`<br><pre style="white-space: pre-wrap;">'Adam',<br>'Eve'</pre> | No<br><pre style="white-space: pre-wrap;">'Eve'</pre>                                      |                                                                                                 |
| `fruit`                             | `array`                                                 | No<br><pre style="white-space: pre-wrap;">[<br>  'Banana'<br>  'Apple'<br>]</pre>          | Possible fruits.                                                                                |
| `alive`                             | `bool`                                                  | No<br><pre style="white-space: pre-wrap;">true</pre>                                       | Indicates if animal is alive                                                                    |
| `age`                               | `int`                                                   | No<br><pre style="white-space: pre-wrap;">3</pre>                                          | Age of animal                                                                                   |
| `animal`                            | `object`                                                | No<br><pre style="white-space: pre-wrap;">{<br>  name: parameters('name')<br>  age: parameters('age')<br>  species: 'Ape'<br>  eats: 'Banana'<br>  alive: parameters('alive')<br>}</pre> |                                                                                                 |
